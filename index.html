    Presentation

    DataShielder est un logiciel de chiffrement de fichier, il intègre des algorithmes tels que le RSA. Le fichier chiffré est généré sous l'extension .ds pour une sécurité maximale. Il y a un système de chiffrment basé sur le RSA et une version beta utilisant les courbes éliptiques. Un générateur de mot de passe est intégré avec 25 charactères pour une sécurité optimale !
